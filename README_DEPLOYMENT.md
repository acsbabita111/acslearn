## Razorpay Setup
Open `payment.html` and replace the placeholder in the Razorpay Payment Button Area with your Razorpay Dashboard Payment Button script.
Never add `key_secret` in public HTML/JS.

## Forms
Forms are static placeholders. Connect them to Google Forms, Firebase, Airtable, Formspree, or your backend.

## Deployment
Upload all files to the GitHub Pages repository root for `acslearn.com`.
