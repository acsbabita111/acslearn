## ढाँचा (v1.2 — इसी दौर में तय)

```
MG_INTRO[mg] = {
  name, body,        ← 🟦 पुराना साझा-ज्ञान (v1.0 से)
  bazar, leaders      ← 🟦 नया साझा (v1.2) — 24 बार रिसर्च, 950 बार नहीं
}
UDYAM_INTRO[n] = {
  slug, title_hi, title_note, mg,
  parts: {
    parichay, outlook, trends, lscale,   ← 🟨 उद्यम-विशेष (हल्का — 1-2 सर्च काफ़ी)
    milte, yogyata, asafal,
    suraksha, safalta, maang,             ← 🟨 उद्यम-विशेष भारी (सुरक्षा/माँग अलग-अलग काम में अलग)
    yojana, gyan, tour, sandesh
  }
}
```

**नियम (generator/build_udyam_intro_pages.js में लागू):** `MG_SHARED_KEYS = ["bazar","leaders"]` —
अगर किसी उद्यम को अपने बाज़ार/बड़े-नाम अलग चाहिए हों (जैसे कोई बहुत ख़ास उप-धंधा), तो
`UDYAM_INTRO[n].parts.bazar` / `.leaders` सीधे लिख दें — वह MG के साझा को override कर देगा।
डिफ़ॉल्ट में कुछ मत लिखें, MG अपने-आप भर देगा।
