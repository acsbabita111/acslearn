ACS Vani Live

Speak in one language — everyone hears and reads in theirs, instantly.

— or join with a code —
Open the same room code on two phones — you'll be connected live. (Internet required)

📖 How to use

Example: three students want to talk to each other — one speaks Bengali, one Kannada, one Marathi. Different languages, yet everyone understands.
  1. All three open this same page on their phones.
  2. Each one picks a role (Student is fine for everyone) and selects their own language — first Bengali, second Kannada, third Marathi.
  3. Any one of them taps "Create new room" and gets a room code (e.g. 5294), then shares that code with the other two.
  4. The other two enter that code and tap "Join room".
  5. Now anyone can speak or type in their own language — everyone else receives it in their own language. The Bengali speaker reads and hears Bengali, the Kannada speaker Kannada, the Marathi speaker Marathi.
Anyone can create or join — both teachers and students get the "Create new room" button.
More than two people can join one room — each with a different language.
Every message shows the sender's language, so even in a group you know who said what.
Choose "Teacher" only when someone is leading a class; for peer conversations everyone can stay a "Student".
